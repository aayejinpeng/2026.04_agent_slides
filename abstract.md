# 摘要

AI 编程工具在 2025-2026 年迎来了爆发式发展，从早期的代码补全（Copilot）快速演进到今天的自主编程 Agent（Claude Code、Codex CLI），开发效率的提升从"锦上添花"变成了"质的变化"。本次分享以**现场 Demo 驱动**的方式，展示两款主流 AI 编程工具——Claude Code（交互式深度开发）、Codex CLI（沙箱化批量任务）的实际使用效果。

分享的核心案例是我们在开题报告调研中构建的 **Survey Pipeline**：一个由 Agent 驱动的全自动化论文调研工具链。该工具从 DBLP 批量抓取 14,600+ 篇论文，经关键词智能打分筛选至 196 篇，再通过 Claude 生成结构化分析、Codex 交叉验证、人工对照审核，最终产出开题报告可直接引用的结构化素材。整个过程将传统 1-2 周的工作量压缩至 1-2 天。

本次分享旨在帮助同学们快速了解当前 AI 编程工具的能力边界，掌握工具选型方法（"写代码用 CC，跑任务用 CX"），并通过我们的 API 服务降低 AI 工具的使用门槛，让每个人都能用上最好的 AI 能力。
