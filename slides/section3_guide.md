## 全景对比

| 项目 | 类型 | 开源 | MCP | 特色 |
|------|------|------|-----|------|
| **Claude Code** | CLI | ❌ | ✅ | Agent 子任务、深度交互 |
| **Codex CLI** | CLI | ✅ | ❌ | 沙箱三级隔离、批量任务 |
| **Cherry Studio** | 桌面 | ✅ | ✅ | 300+ 模型、零门槛 |
| **Cursor** | IDE | ❌ | ✅ | Tab 补全、Agent Mode |
| **GitHub Copilot** | IDE | ❌ | ❌ | 生态最广 |



## 什么时候用什么？

| 场景 | 推荐 | 为什么 |
|------|------|--------|
| 写代码、改 Bug、重构 | **Claude Code** | 深度理解代码库，交互式修改 |
| 批量自动化、安全审查 | **Codex CLI** | 沙箱隔离，适合无人值守 |
| 日常 AI 对话、多模型 | **Cherry Studio** | 图形界面友好，300+ 模型 |
| 论文阅读、翻译 | **Cherry Studio** | 知识库 RAG，文档导入 |
| 项目级复杂 Agent | **Claude Code** | Agent 子任务分发 |
| 简单脚本、一次性任务 | **Codex CLI** | 一条命令搞定 |
| 用我们的 API | **Cherry Studio** | 配 Key 即用，零门槛 |
